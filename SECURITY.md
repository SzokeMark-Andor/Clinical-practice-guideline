# Security Policy

## Supported Versions
This is a research / quality-improvement software artifact.
Security fixes (if any) are applied on a best-effort basis to the default branch.

## Reporting a Vulnerability
If you discover a security issue, please report it privately:

- Email: szoke_52@yahoo.com

Please include:
- A clear description of the issue
- Steps to reproduce
- Impact (what an attacker could do)
- Any logs/screenshots that help

**Please do not open a public GitHub issue for security reports.**

## Responsible Disclosure
Please allow time for investigation and a fix before public disclosure.
If your report involves any dataset, share only anonymized/minimal reproductions.
